## 一、项目业务背景与核心痛点
### 1.1 业务背景
传统行业投资分析普遍存在顺周期思维惯性：市场习惯在行业景气上行、股价上涨阶段看多加仓，最终出现利好完全兑现、高位入场亏损的问题。

资本市场存在稳定逆向规律，以航空行业为例：
1. 极致利空阶段（2023疫情航空）：行业基本面亏损触底，但悲观预期提前全部定价，股价率先企稳见底，形成高安全边际逆向买点；
2. 极致利好阶段（当前航空周期高位）：行业盈利修复至历史峰值，但乐观预期透支，股价持续走弱，构成逆向卖出信号。

强周期行业供需、盈利、政策波动幅度大，基本面与市场价格长期错位，是逆向投资策略最优适配赛道，因此本系统仅针对周期行业做标准化量化分析。

### 1.2 人工分析现存痛点
1. 判断主观化：底部、顶部拐点依赖个人投研经验，无统一可量化标准；
2. 数据碎片化：无法长期统一跟踪景气、估值、行情、情绪四维背离关系；
3. 判断滞后：易受短期新闻、日内行情干扰，错过周期极值拐点；
4. 筛选低效：全市场行业数量庞大，无法快速聚焦具备逆向价值的周期赛道。

### 1.3 项目业务目标
搭建一套标准化、可复盘、可自动化执行的周期行业逆向分析业务体系，实现：
1. 量化识别周期行业中基本面与市场预期背离带来的逆向投资机会；
2. 统一逆向买点、卖点、观望区间的标准化判定规则；
3. 固定高弹性周期行业分析池，过滤无逆向分析价值的弱周期/稳定行业；
4. 输出客观量化结论，消除人工主观判断偏差。

## 二、核心业务全局定义
### 2.1 目标分析行业筛选标准
系统仅纳入**强周期性行业**，行业需同时满足两项条件：
1. 景气具备完整「萧条-复苏-繁荣-衰退」四轮周期轮动特征；
2. 盈利、供需、产品价格、营收波动幅度大，易出现极端高估/低估。

#### 系统固定周期行业分析池
航空机场、生猪养殖、基础化工、煤炭、有色（铜/铝）、远洋航运、酒店旅游、周期半导体、光伏上游原材料、工程机械

### 2.2 逆向投资底层核心公理（不可修改）
所有逆向投资机会仅产生于**基本面与市场定价背离**，分为两类核心场景：
1. 逆向买点背离：基本面跌至历史最差区间，但行业指数、估值不再同步创新低（利空充分定价）；
2. 逆向卖点背离：基本面升至历史最优区间，但行业指数、估值持续走弱（利好完全透支）；
3. 补充规则：基本面与股价同向波动时，属于趋势行情，不参与逆向布局。

### 2.3 四大统一分析维度（全行业通用）
所有周期行业统一使用四维指标做背离校验，计算口径全局统一：
1. **行业景气度维度**：行业真实经营基本面，作为投资核心锚点；
2. **行业估值维度**：行业整体PE-TTM、PB水位，反映市场资产定价泡沫/折价；
3. **行业价格收益维度**：行业指数中长期涨跌幅，代表全市场交易者一致预期；
4. **市场情绪维度**：机构持仓、行业舆情、一致盈利预测，验证市场情绪是否到达极值。

## 三、标准化业务判定规则
### 3.1 统一量化标尺：近10年历史分位（0~100）
所有维度原始数据统一换算为近10年历史百分位，分档标准固定：
- 0% ~ 20%：历史极致底部区间
- 20% ~ 80%：历史中性震荡区间
- 80% ~ 100%：历史极致顶部区间

### 3.2 逆向买入机会判定（全部条件同时满足方可标记机会）
核心逻辑：行业基本面极致恶化，但市场预期提前修复，利空出清
1. 景气度分位 ≤ 20%：行业经营处于近十年最差区间；
2. 景气边际改善：当期景气环比降幅收窄或小幅转正，基本面不再持续恶化；
3. 估值水位分位 ≤ 25%：行业估值处于历史低位，泡沫完全出清；
4. 核心背离条件：行业价格分位 ＞ 景气度分位；释义：基本面已创阶段新低，但股价未同步创新低，悲观预期充分定价；
5. 情绪校验：机构持仓、行业负面舆情处于阶段极值，市场整体一致看空；
6. 风控前置校验：通过价值陷阱过滤规则。

### 3.3 逆向卖出风险判定（任意2条触发即提示卖出预警）
核心逻辑：行业基本面达到周期顶峰，但市场乐观预期提前透支见顶
1. 景气度分位 ≥ 80%：行业盈利、供需景气处于近十年顶部；
2. 核心背离条件：景气持续创新高，但近3月/12月行业指数持续下跌；
3. 估值条件：估值从历史高位持续回落，盈利增速无法支撑当前估值；
4. 情绪条件：行业政策、需求利好密集落地，市场一致看多，散户资金持续流入。

### 3.4 观望区间判定（无逆向布局价值）
满足任意一条即划入观望池：
1. 景气度分位处于20%~80%中性区间，无周期极值特征；
2. 基本面走势与行业指数涨跌同向，无明显背离关系；
3. 周期底部/顶部拐点未确认，景气边际变化模糊。

### 3.5 风控过滤规则（规避价值陷阱）
满足买入全部条件后，必须通过以下风控校验，否则直接剔除机会池：
1. 排除长期政策永久压制、需求持续萎缩的夕阳行业；
2. 排除行业长期产能过剩、格局永久性崩塌赛道；
3. 排除大规模资产减值、持续亏损无修复预期的行业。

## 四、标准化业务分析执行流程
系统每次执行行业分析，严格遵循固定流程，不可调整顺序：
1. 加载行业池：仅读取预设强周期行业列表，跳过其余行业；
2. 四维数据更新：拉取景气、估值、价格、情绪最新周期数据；
3. 历史分位计算：全部指标统一换算为近10年百分位；
4. 背离关系校验：对比基本面与估值、指数走势是否出现反向背离；
5. 买卖信号匹配：对照买入、卖出、观望规则完成初步标记；
6. 风控二次过滤：剔除存在永久经营风险的行业；
7. 逆向得分计算：生成0-100分逆向价值分数；
8. 标准化输出：输出行业完整分析报告与最终操作结论。

## 五、业务输出规范
### 5.1 逆向价值综合得分区间定义（0~100分）
- 0 ~ 30分：无逆向投资价值，建议规避；
- 30 ~ 60分：周期底部观察区间，持续跟踪景气边际变化；
- 60 ~ 100分：高确定性逆向投资机会，重点跟踪观察。

### 5.2 标准输出字段清单
1. 行业名称；
2. 近10年景气分位、估值分位、价格分位；
3. 基本面边际变化标签（改善/持平/恶化）；
4. 文字化背离逻辑解读（用于人工复盘）；
5. 逆向综合得分；
6. 最终业务结论：【重点买入跟踪 / 持续观望 / 风险卖出】；
7. 专属风险提示（周期波动、政策、上游成本、供需风险）。

## 六、基准业务案例校准（航空行业）
### 6.1 2023年航空行业（标准逆向买点样本）
1. 景气分位：5%，处于历史极致亏损底部；
2. 估值分位：20%，历史极低估值区间；
3. 价格分位：30%，价格分位显著高于景气分位，强背离成立；
4. 边际变化：单季度亏损持续收窄，基本面边际改善；
5. 风控校验：无永久性需求萎缩风险；
6. 结论：高得分逆向买入机会。

### 6.2 当前航空行业（标准逆向卖点样本）
1. 景气分位：85%，行业盈利处于近十年高位；
2. 价格走势：景气持续向好，但行业指数中长期震荡下行；
3. 市场情绪：复苏利好全部兑现，市场乐观预期充分透支；
4. 结论：多条卖出条件触发，提示规避。

## 七、系统业务边界约束（永久固定）
1. 分析对象边界：仅分析行业整体、行业指数、行业汇总基本面，不处理任何单只个股数据；
2. 行业范围边界：仅使用预设周期行业池，不自动遍历全市场所有行业；
3. 程序形态边界：仅本地Python分析程序，无小程序、无云端服务、无可视化前端页面；
4. 策略边界：仅执行背离逆向分析，不开发趋势跟踪、成长选股、短线波段策略；
5. 输出边界：仅输出量化分析结论与风险提示，不生成、推送任何实盘交易指令。
